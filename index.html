Founder Agent OS

Local-first Agent OS — runtime for human-AI collaboration

Mosaic is a local-first Agent OS — a runtime for human-AI collaboration where your data stays on your machine. Connect models, compose agents, and inspect every run with typed decision replay, graph-native memory, and multi-provider workflows. Agents get real tools via MCP integration with Claude Code and Cursor, backed by SQLite-based knowledge graphs — all without sacrificing privacy or control.
